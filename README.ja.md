# Disco Science Lite

_The Disco Must Grow. 宇宙へ、そしてパフォーマンスの進化とともに。_

Disco Science Lite は、Daniel Brauer 氏の人気 Mod [Disco Science](https://mods.factorio.com/mod/DiscoScience) の非公式派生版です。アルゴリズムによる UPS 最適化と、Space Age Mod 対応などの追加機能を備えています。

**マルチプレイヤー対応はまだ実験的です。ゲーム参加時の desync エラーや、UPS 負荷を増大させる恐れがあります。大人数でのゲームや、重要なゲームへの導入は避けてください。**

この Mod は原作者とは関係ありません。この Mod に関する問い合わせやバグ報告を、原作者や原作の Discussion に送らないでください。

## 機能

研究所が、消費しているサイエンスパックの色で、ディスコのように光り輝きます。工場を一目見るだけで、何を研究中かがひと目でわかります。

## 特徴

- **パフォーマンス**: UPS（Updates Per Second）への影響を最小限に抑えるよう大幅に最適化されており、大規模な工場で大量のラボがあっても快適に動作します。Mod 設定から追加のパフォーマンスチューニングも可能です。（[技術的詳細:en](docs/ups-optimization.md)）

- **組み込み Mod サポート**: Space Age のバイオ研究所、惑星追加 Mod（Maraxsis、Cerys など）、Krastorio 2、Exotic Space Industries など、20 以上の Mod が専用のカラーアニメーション付きでサポートされています。全リストは下記を参照してください。

- **未対応の研究所の自動着色**: 未対応の Mod が追加した研究所も 汎用的な発光エフェクトで自動的に着色します。Mod 設定でオン・オフを切り替えられます。

- **色のカスタマイズ**: Mod 設定から色の彩度や明度を調整して、好みの色に設定できます。

- **新しい色パターン**: 原作 Mod のパターンに加え、異なる視覚効果を持つ新しい色パターンを追加しています。

- **Mod API 互換性**: 原作の Disco Science の API と互換性があるため、Disco Science に対応している大半の Mod がそのまま動作します。

## 互換性

この Mod はゲーム途中での追加・削除が安全に行えます。

原作の Disco Science を使用している場合は、Disco Science Lite を使用するために無効化する必要があります。
