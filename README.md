# Disco Science Lite

## Development

Requirements: `lua`, `luarocks`

To install dependencies by luarocks:

```
make dev
```

To lint:

```
make lint
```

To run unit tests:

```
make test
```

To update image files (requires ImageMagick, Factorio, Space-Age DLC):

```
make graphics
```
